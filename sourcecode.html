List of working versions

The hydrodynamic model started as a Fortran code in serie for estuaries, which was later modified to be suitable for lake studies. Over the time, researchers have contributed in the evolution of Si3D by adding a water quality module and by modifying the source code to work in parallel as investigations required greater computational requirements. Due to the variable versions of Si3D, we provide the GitHub link to two of the main master source codes available. Please click on the GitHub icon to access the source code of your choice.

Master version in serie (Si3D-series)

This version of the code can be found in the Si3Dl GitHub website as a repository with the name of si3d-series. The source code includes the water quality module ECOMOD and the nested grid utility where finer meshes can be used in local areas of the domain.

Si3D-series

Master version in parallel (Si3D)

The version in parallel is a modification to the original source code by including multithread optimization within the modules of the hydrodynamical model. This parallelization of the code in serie was developed by Ph.D. Mario Acosta. This version also includes the water quality model and the nested grid capability.
